The Maze Game
================

An implementation for the popular Maze Game 

Using Ring Game Engine for 2D Games 

Wikipedia Page : https://en.wikipedia.org/wiki/Maze

	Arrows (Up|Down|Right|Left) : Move the Player

	ESC : Close the Game
	
	SPACE : Restart the level
