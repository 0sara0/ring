=================================
Control Structures - Second Style
=================================

In this chapter we are going to learn about the second style of control structures
provided by the Ring programming language.

.. index:: 
	pair: Control Structures; Branching


Branching
=========

* If Statement

Syntax:

.. code-block:: none

	if Expression
		Block of statements
	elseif Expression
		Block of statements
	else
		Block of statements
	end

Example:

.. code-block:: none

	put " 
		Main Menu
		---------
		(1) Say Hello
		(2) About
		(3) Exit

	    " get nOption

	if nOption = 1	put "Enter your name : " get name put "Hello " + name + nl
	elseif nOption = 2 put "Sample : using if statement" + nl
	elseif nOption = 3 bye
	else put "bad option..." + nl
	end


* Switch Statement

Syntax:

.. code-block:: none

	switch Expression
	case Expression
		Block of statements
	else
		Block of statements
	end		

Example:

.. code-block:: none

	Put " 
		Main Menu
		---------
		(1) Say Hello
		(2) About
		(3) Exit

	    " Get nOption

	Switch nOption
	Case 1 Put "Enter your name : " Get name Put "Hello " + name + nl
	Case 2 Put "Sample : using switch statement" + nl
	Case 3 Bye
	Else Put "bad option..." + nl
	End

Looping
=======

* While Loop

Syntax:

.. code-block:: none

	while Expression
		Block of statements
	end


* For Loop

Syntax:

.. code-block:: none

	for identifier=expression to expression [step expression]
		Block of statements
	end

* For in Loop

Syntax:

.. code-block:: none

	for identifier in List/String  [step expression]
		Block of statements
	end
